## 🚀 What You Need to Do (3 Steps, 20 Minutes)

### Step 1: Get MongoDB (5 minutes)
Choose one:
- **Option A:** Create free account at https://cloud.mongodb.com
- **Option B:** Use local MongoDB installation

Get connection string → Copy to clipboard

### Step 2: Configure Backend (2 minutes)
```bash
# Edit backend/.env and add:
MONGODB_URL=your_connection_string
JWT_SECRET=generate_with_command_below
JWT_EXPIRATION_HOURS=24
JWT_ALGORITHM=HS256

# Generate JWT_SECRET:
python -c "import secrets; print(secrets.token_urlsafe(32))"
```

### Step 3: Install & Run (13 minutes)
```bash
# Install dependencies
cd backend && pip install -r requirements.txt
cd ../frontend && npm install

# Terminal 1:
cd backend
python -m uvicorn app.main:app --reload --port 8000

# Terminal 2:
cd frontend
npm run dev

# Then open: http://localhost:3000
```

## 🎓 How It Works (Simple Explanation)

### User Signup
```
User enters email/password/name
        ↓
Backend hashes password with bcrypt
        ↓
User saved to MongoDB
        ↓
JWT token generated (valid 24 hours)
        ↓
User logged in, redirected to dashboard
```

### User Login
```
User enters email/password
        ↓
Backend verifies password hash
        ↓
JWT token generated
        ↓
Token sent to frontend, stored in localStorage
        ↓
User can access dashboard
```

### Using the App
```
User clicks "Upload Bill" or views expenses
        ↓
Frontend attaches JWT to request
        ↓
Backend validates JWT
        ↓
Extract user_id from JWT
        ↓
Query MongoDB: SELECT * WHERE user_id = current_user_id
        ↓
Return ONLY user's data
        ↓
Display on dashboard
```

### Token Expiration
```
Token reaches 24-hour limit
        ↓
Frontend gets 401 Unauthorized
        ↓
Token removed from localStorage
        ↓
Redirect to login page
        ↓
User must login again
```

## 🗄️ Database Schema

### Three Collections (Created in MongoDB)

**users:**
- email (unique)
- full_name
- password_hash (bcrypt encrypted)
- created_at, updated_at

**expenses:**
- user_id (which user owns this)
- bill_name
- vendor
- category
- amount
- gst
- date
- created_at, updated_at

**categories:**
- user_id (which user owns this)
- name
- created_at

**Every query includes:** `WHERE user_id = current_user_id`

## 🔧 What Happens at Each Step

| Action | Backend Response | Frontend Behavior |
|--------|------------------|-------------------|
| Sign up | Creates user, returns JWT | Stores JWT, redirects to dashboard |
| Log in | Validates password, returns JWT | Stores JWT, redirects to dashboard |
| Request data | Validates JWT, filters by user_id | Shows user's data only |
| Token expires | Returns 401 Unauthorized | Clears JWT, redirects to login |
| Log out | Endpoint called | Clears JWT, redirects to login |
